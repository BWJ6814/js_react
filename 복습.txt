# map()
배열을 반복하면서 각 요소를 꺼냄 그리고 콜백함수가 return한 값들로
이루어진 새 배열을 반환
const products = [
  { id: 1, name: "사과" },
  { id: 2, name: "바나나" }
];
const result = products.map((item) => {
  return item.name;
});
결과값 ["사과", "바나나"]
