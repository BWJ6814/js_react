핵심개념
1. 컴포넌트: 화면을 이루는 재사용 가능한 UI 조각.. 함수하나가 화면 조각을 리턴한다고 생각하자!
2. JSX : 자바스크립트안에서 HTML 비슷한 문법을 쓰게 해주는 문법.. 결국 JS로 변환된다..
3. PROPS : 부모 컴포넌트가 자식 컴포넌트에게 전달하는 데이터(읽기전용) "속성" 처럼 생각하자,,
4. STATE : 컴포넌트 자체가 들고 있는 바뀌는 값. useState로 관리함
5. 단방향 데이터 흐름 : 데이터는 부모 -> 자식(props) 방향으로 내려감(자식에서 부모 값을 바꾸고 싶으면
   부모가 준 콜백함수를 호출)

외워둘 3가지 규칙
1. props는 읽기 전용(불변) → 자식이 직접 수정하면 안 됨
2. state는 컴포넌트 내부에서만 바꿈 (setState 계열 함수로)
3. 데이터는 위에서 아래로 흘러감(부모→자식). 자식이 부모 걸 바꾸고 싶으면 부모 콜백을 호출
